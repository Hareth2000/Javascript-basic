## Q1: Evaluate the Expressions

2. `"JavaScript " + 50` => `"JavaScript 50"`
3. `17 % 5` => `2`
4. `5 % 17` => `5`
5. `5 / 10` => `0.5`
6. `(4 === '4')` => `false`
7. `(4 != 5)` => `true`
8. `(7 <= 8)` => `true`
9. `"Hello" + 5` => `"Hello5"`
10. `Math.ceil(x) - Math.floor(x)` => `0` or `1`
11. `Math.pow(x, 2)` => \(x^2\)
